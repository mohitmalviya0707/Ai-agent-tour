# Ai-agent-tour

# 🎧 AI Tour Guide (Multi-Agent GenAI System)

An advanced Generative AI project that creates personalized audio tours using multiple AI agents.

## 🚀 Features

- Multi-Agent Architecture (Planner, Architecture, History, Culture, Culinary)
- Real-time Web Search Integration
- Personalized Tour Generation
- Async Execution for Performance
- Rich CLI UI with live progress

## 🧠 Agents Used

- Planner Agent → decides time allocation
- Architecture Agent → buildings & design
- History Agent → historical storytelling
- Culture Agent → traditions & lifestyle
- Culinary Agent → food & restaurants
- Orchestrator Agent → final tour creation

## ⚙️ Tech Stack

- Python
- OpenAI Agents SDK
- Pydantic
- Rich (CLI UI)
- Asyncio
